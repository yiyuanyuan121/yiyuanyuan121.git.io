3.快速入门

优点

  • 内容和表现分离
  • 网页结构变现统一，可以实现复用
  • 样式十分丰富
  • 建议使用独立的css文件
  • 容易被搜索引擎收录

vue框架不容易被搜索引擎收录

4.css的三种导入方式

  1. 内部样式

    • <!--内部样式-->
<style>
    h1{
        color: brown;
    }
</style>

  2. 行内样式

    • <!--行内样式：在标签中写入一个style，然后在style中编写声明-->
<h1 style="color: blue"></h1>

    导入式的弊端：如果网页太大，那么就会先展现出结构，之后才会出现样式

5.选择器

选择页面中的某一个，或者某一类元素

5.1基本选择器

  • 标签选择器

  • 类选择器

  • id选择器

    • /*id选择器
【格式】 #id名称{}
id是唯一的
*/
#test{
    color: #5cff96;
}
<p  id="test">自己学</p>

区别：

标签选择器：作用于标签：可以用来格式化网页

class 选择器：作用于class ;可以复用

id选择器：全局唯一

优先级：

id>class>标签
