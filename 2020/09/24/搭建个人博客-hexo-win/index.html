搭建个人博客-hexo-win

注意：提前安装Git

Git连接：https://git-scm.com/

hexo 是需要node支持的，所以要安装node.

1.下载node.js

下载地址https://nodejs.org/zh-cn/

**2.**安装node.js

(安装比较简单 ，一律next就可以了）

验证方法一

安装好以后，使用win+R快捷键，输入cmd或者node打开DOS窗口输入node -v进行验证。

验证方法二

打开node（在电脑菜单栏能看到）

点击Node.js command prompt 执行Node.js

可以输入npm -v 查看自己包管理器的版本（非必须操作）

3、安装cnpm

我们是利用npm【包管理器】进行安装cnpm，但是国内的镜像源比较慢，所以利用npm安装cnpm【淘宝的】

输入npm install -g cnpm –registry=https://registry.npm.taobao.org (利用npm安装cnpm,借助淘宝的npm进行镜像源安装 -g 是全局安装 registry是镜像源 ) 等待安装完全以后，可以使用cnpm -v进行验证

5.**利用cnpm安装hexo**

然后可以使用hexo -v 进行验证

**6.**建立一个空的文件夹

然后输入hexo init (如果没有安装Git会报错 )【注意：要在你的空文件夹路径下】

**7.**启动博客

，输入hexo s 启动（其中的http://localhost:4000是你的博客本地窗口地址，起到预览作用）
